RegExp Object - A RegExp Object is a pattern with Properties and Methods

Component Descrpition
1. ignoreCase The ignoreCase property specifies whether or not the "i" modifier is set. This property returns true if the "i" modifier is set, otherwise it returns false.
2. multiline The multiline property specifies whether or not the m modifier is set. This property returns true if the "m" modifier is set, otherwise it returns false.
3. source The source property returns the text of the RegExp pattern.
4. test() The test() method tests for a match in a string. If it finds a match, it returns true, otherwise it returns false.
5. exec() The exec() method tests for a match in a string. If it finds a match, it returns a result array, otherwise it returns null.

RegExp Modifiers - A RegExp Modifiers can be used to perform case-insensitive more global searches

Component Descrpition
1. g modifier The "g" modifier specifies a global match. A global match finds all matches (compared to only the first).
2. i modifier The "i" modifier specifies a case-insenitive match.
3. m modifier The "m" modifier specifies a multiline match. It only affects the behavior of start ^ and end $. ^ specifies a match at the start of a string. $ specifies a match at the end of a string. With the "m" set, ^ and $ also match at the beginning and end of each line.

RegExp Group/Brackets - Brackets can define single characters, groups, or character spans

Component Descrpition
1. [A-z] Brackets [abc] specifies matches for the characters inside the brackets. Any character from uppercase A to lowercase z
2. [^abc] Brackets [^abc] specifies matches for any character NOT between the brackets. Brackets can define single characters, groups, or character spans
3. [^0-9] The [^0-9] expression is used to find any character that is NOT a digit. The digits inside the brackets can be any numbers or span of numbers from 0 to 9.
4. (x|y) The (x|y) expression is used to find any of the alternatives specified. The alternatives can be of any characters.

RegExp Metacharacters - The RegExp Metacharacter in JavaScript is used to find the whitespace characters.
The whitespace character can be a space/tab/new line/vertical character.

Component Descrpition
1. \w The \w metacharacter matches word characters. A word character is a character a-z, A-Z, 0-9, including _ (underscore).
2. \D The \D metacharacter matches non-digit characters.
3. \0 The \0 metacharacter maches NUL characters.
4. \v The \v metacharacter matches vertical tab characters (tabulators).
5. \udddd The \udddd metacharacters matches Unicode characters specified by a hexadecimal number (dddd).

RegExp Quantifiers - The RegExp Quantifiers match a number of instances of a character, group, or character class in a string.

Component Descrpition
1. * Matches zero or more occurrences of the preceding element
2. + Matches one or more occurrences of the preceding element
3. ? Matches zero or one occurrence of the preceding element
4. {n} Matches exactly n occurrences of the preceding element
5. {n,} Matches at least n occurrences of the preceding element
6. {n,m} Matches at least n and at most m occurrences of the preceding element

RegExp methods - The RegExp method are used to find matches for regexp in the string str.

Component Descrpition
1. test() Tests for a match in a string. Returns true or false.
2. exec() Searches for a match in a string. Returns an array of information about the match or null if no match was found.
3. match() Searches for a match in a string. Returns an array of information about the match or null if no match was found.
4. replace() Replaces a substring in a string with a new substring.
5. search() Searches for a match in a string. Returns the index of the match or -1 if no match was found.
